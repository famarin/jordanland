# أراضي الأردن — Jordan Land

منصة متكاملة لبيع الأراضي في المملكة الأردنية الهاشمية

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Deployment

This project auto-deploys to GitHub Pages when you push to the `main` branch.

Your site will be live at: `https://YOUR-USERNAME.github.io/jordan-land/`
